BAB 1: PENGANTAR HTML

Apa itu HTML

HTML adalah singkatan dari HyperText Markup Language. Secara sederhana, HTML adalah cara untuk memberi struktur pada sebuah halaman web dengan menggunakan berbagai macam tag atau elemen markup. Tag-tag ini memberi petunjuk kepada browser bagaimana cara menampilkan informasi di halaman web, seperti teks, gambar, video, tabel, dan lainnya.

HTML bukanlah bahasa pemrograman, melainkan bahasa markup. Itu berarti HTML hanya digunakan untuk memberi struktur dan format pada halaman, bukan untuk menjalankan logika atau perhitungan seperti yang dilakukan bahasa pemrograman (seperti Python atau JavaScript).

Kenapa HTML Itu Penting?

HTML adalah dasar dari semua halaman web. Tanpa HTML, halaman web tidak akan bisa ditampilkan. Misalnya, jika kamu mengunjungi sebuah halaman web, browser kamu akan membaca HTML untuk mengetahui apa yang harus ditampilkan dan bagaimana menampilkannya.

Dasar-Dasar Struktur HTML

Sebuah dokumen HTML memiliki struktur yang konsisten dan harus mengikuti aturan tertentu agar browser bisa membacanya dengan benar. Setiap dokumen HTML dimulai dengan sebuah deklarasi <!DOCTYPE html> yang memberitahukan browser bahwa dokumen ini menggunakan HTML5, versi terbaru dari HTML.

Berikut adalah struktur dasar dari sebuah dokumen HTML:


<!DOCTYPE html>
<html lang="id">
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Judul Halaman</title>
  </head>
  <body>
      <h1>Selamat Datang di Halaman HTML</h1>
      <p>Ini adalah paragraf pertama saya di HTML.</p>
  </body>
</html>

Ini adalah deklarasi yang memberi tahu browser bahwa dokumen ini adalah HTML5. HTML5 adalah standar terbaru yang digunakan dalam pengembangan web modern.

2.<html>

Tag ini menandai awal dari dokumen HTML dan mencakup seluruh konten halaman. Atribut lang="id" memberitahu browser bahwa bahasa yang digunakan dalam halaman ini adalah bahasa Indonesia.
